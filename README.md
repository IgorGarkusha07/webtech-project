# CyberVerse - Tech Store Project
This is my project for webtech
## Project structure
* index.html - Home page
* catalog.html - Catalog of devices
* product.html - Detailed description of one phone
* gallery.html - Photos of devices and description
* contact.html - Contact information and form for feedback
* blog.html - Section with news
* style.css - CSS file with all design and style of my project
* script.js - JavaScript file with AJAX system to load table 
* data.json - JSON file database for AJAX

## Starting of my project
### Option 1: direct launch
1. download all files in one directory
2. start "index.html"

### Option 2: WeBstorm (recommended)
1. download all files in one directory
2. open index.html
3. choose chrome on the right side 
4. start project
